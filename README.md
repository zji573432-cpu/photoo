## 🎯 photoo 是什么？

**photoo** 是一款 Android 原生的极简照片整理工具。

滑动卡片就能快速整理相册——左滑删除，右滑收藏，上滑保留。还有一系列"该有的全都有"的功能：相册分析生成专属摄影画像、沉浸式回顾喜爱照片、智能识别相似照片和低质量图片、视频和图片压缩......

**但 photoo 不只是一个"清理"软件，它更是"回忆"。**

---

## ✨ 核心亮点

### 👆 滑动卡片整理
和刷短视频一样有趣。卡片顺序默认随机，你永远不知道下一张是什么——可能是上周拍的，也可能是一年前拍的。保持好奇，也就有了整理的动力。

* 左滑删除，右滑收藏，上滑保留
* 卡片底部显示拍摄时间、地点等 EXIF 信息，还有端侧 AI 识别的拍摄内容
* 一组 16 张（可在设置调整为 10-30 张），把成千上万张照片变成轻松的"微任务"

### � 相册分析
快速生成独属于你的摄影画像。了解自己的拍摄习惯，记录每一次按下的快门。

### 🔍 智能识别
* **相似照片**：自动找出重复或极其相似的照片
* **低质量图片**：智能识别模糊、暗光、过曝等废片
* **视频/图片压缩**：画质几乎不变，文件轻松小一半

### 💾 后悔药
* 支持撤销操作
* 删除前可调整，误删可能性降至最低

### 🔒 隐私至上

**这是 photoo 的底线，也是和其他产品最大的区别。**

相册是绝对隐私的存在，因此 photoo 设计之初，一切的原点是**隐私保护**：

* **零联网**：完全不需要网络权限
* **零数据上传**：所有分析在本地完成
* **100% 本地运行**：人脸检测、图像分类全在手机端

为了实现这个目标，我们做了一个大胆的决定：**完全放弃广告、内购和订阅收入**——因为这些往往意味着需要联网和接入第三方 SDK。

**photoo 将永远免费、无广告、无需注册和登录、可以不联网使用。**
